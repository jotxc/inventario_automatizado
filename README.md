# Projeto de Automação de Inventário com Excel e Power Query
Este projeto tem como objetivo automatizar e organizar o processo de inventário de um armazém, reduzindo tarefas manuais e melhorando o controle de lotes e posições.

Tecnologias utilizadas

Microsoft Excel

Power Query (M Language)

(Futuro) Python / VBA / Sistema Web

Funcionalidades atuais

Filtragem automática de lotes pendentes

Consolidação de bases de inventário

Organização por ruas (EA, EB, EC, etc.)

Renumerar índices após filtros

Redução de retrabalho operacional

Roadmap do projeto

- Otimizar performance das consultas do Power Query

- Criar dashboard de acompanhamento

- Automatizar geração de listas de contagem

- Desenvolver sistema próprio para inventário

- Integrar banco de dados e API

Objetivo profissional

Este é meu primeiro projeto profissional voltado para automação de processos logísticos e análise de dados, servindo como base para evolução futura em desenvolvimento de sistemas e engenharia de dados.
