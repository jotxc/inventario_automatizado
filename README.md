# Projeto de Automação de Inventário com Excel e Power Query
Projeto criado a partir de um problema real identificado durante operações de inventário em estoque. O objetivo é reduzir atividades manuais, melhorar a confiabilidade das contagens e evoluir gradualmente a solução até um sistema completo de apoio operacional.

## Contexto

Durante a rotina de inventário em um estoque de materiais produtivos, identifiquei diversas atividades manuais que consumiam tempo e aumentavam a possibilidade de erros operacionais.

O processo envolvia a seleção de lotes, controle de posições já contadas, acompanhamento de pendências e organização das contagens por ruas do estoque, exigindo consultas e verificações repetitivas.

Diante desse cenário, surgiu a ideia de desenvolver uma solução utilizando Excel e Power Query para automatizar etapas do processo e fornecer informações mais rápidas e confiáveis para a operação.

## Problema

Principais dificuldades identificadas:

- Controle manual das contagens realizadas.
- Dificuldade para identificar pendências rapidamente.
- Necessidade de consolidar informações de múltiplas fontes.
- Tempo elevado gasto em tarefas repetitivas.
- Risco de erros operacionais por manipulação manual dos dados.

## Visão do Projeto

O objetivo não é apenas criar uma planilha de apoio ao inventário.

A proposta é evoluir gradualmente a solução, passando pelas seguintes etapas:

1. Automatização com Excel e Power Query.
2. Criação de indicadores e dashboards.
3. Desenvolvimento de automações complementares.
4. Construção de um sistema próprio para gestão de inventários.
5. Possível integração com bancos de dados e sistemas corporativos.

Este repositório documenta toda essa evolução.

## Tecnologias utilizadas

- Microsoft Excel
- Power Query (M Language)
- (Futuro) Python / VBA / Sistema Web

## Funcionalidades atuais

- Filtragem automática de lotes pendentes
- Consolidação de bases de inventário
- Organização por ruas do estoque
- Renumerar índices após filtros
- Redução de retrabalho operacional

## Roadmap do projeto

- Otimizar performance das consultas do Power Query
- Criar dashboard de acompanhamento
- Automatizar geração de listas de contagem
- Desenvolver sistema próprio para inventário
- Integrar banco de dados e API
